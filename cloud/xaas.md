# Everything as a Service

## Infrastructure as a Service (*IaaS*)
- servers
- storage
- networking
- virtualization

e.g. AWS

## Platform as a Service (*PaaS*: IaaS + OS + runtime)

e.g. Kubernetes, AWS Elastic Beanstalk, Heroku

## Software as a Service (*SaaS*: PaaS + apps + data)

e.g. Dropbox
