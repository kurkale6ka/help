# Everything as a Service

## Infrastructure as a Service (**IaaS**)
- servers
- storage
- networking
- virtualization

e.g. AWS

## Platform as a Service (**PaaS**: _IaaS_ + OS + runtime)

e.g. Kubernetes, AWS Elastic Beanstalk, Heroku

## Software as a Service (**SaaS**: _PaaS_ + apps + data)

e.g. Dropbox
