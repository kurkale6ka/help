                                      Sort
Vim
---
                  .- do a numeric sort
                 /
:sort! /\d\+$/ rn
      \         \
       \         *-- sort on the pattern (or after if no r)
        *- reverse

GNU
---
             .- sort on the 3rd field
            /    .-- do a numeric sort
           /    /
sort -t: -k3 -rn
       \       \
        \       *-- reverse
         *- separator for -k
